Built for the Modern Web

Every block is crafted with attention to detail, following WordPress best practices and web standards.

  • Native HTML elements — Accordions use <details>/<summary> for zero-JS functionality
  • Lazy loading & facades — Images and videos load on demand for faster pages
  • Syntax highlighting — 25+ languages supported in the Code block
  • Consistent naming — All blocks use acf- prefix to avoid CSS conflicts
  • AJAX interactions — Star rating submissions without page reload
// No configuration needed
// Just activate the plugin!

register_block_type(
  'acf/accordion',
  ['render_template' => '...']
);

// Field groups auto-load from JSON
acf_add_local_field_group(
  json_decode($json)
